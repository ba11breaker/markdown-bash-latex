# Learning Plan: Markdown, Bash & LaTeX

## Overview

This workspace is organized into three tracks. Each track progresses from
fundamentals to real-world projects. Work through them in order, or jump
between tracks as needed.

---

## Track 1: Markdown (3-4 weeks)

| Week | Module | Folder | Topics |
|------|--------|--------|--------|
| 1 | Basics | `markdown/01-basics/` | Headings, paragraphs, emphasis, links, images, line breaks |
| 1 | Extended | `markdown/02-extended/` | Tables, task lists, footnotes, definition lists, code blocks |
| 2 | GitHub-Flavored | `markdown/03-github-flavored/` | GFM alerts, math, autolinks, emoji, diff syntax |
| 2-3 | Mermaid Diagrams | `markdown/04-mermaid-diagrams/` | Flowcharts, sequence, class, state, ER, Gantt, git graph, mind maps, styling |
| 3-4 | Projects | `markdown/05-projects/` | README, blog post, documentation site, cheat sheet |

## Track 2: Bash (4-6 weeks)

| Week | Module | Folder | Topics |
|------|--------|--------|--------|
| 1 | Basics | `bash/01-basics/` | Commands, navigation, pipes, redirects, permissions |
| 1-2 | Variables | `bash/02-variables-and-types/` | Variables, strings, arrays, arithmetic, quoting |
| 2-3 | Control Flow | `bash/03-control-flow/` | if/else, case, for, while, until, select |
| 3 | Functions | `bash/04-functions/` | Defining functions, return values, local vars, recursion |
| 4 | Text Processing | `bash/05-text-processing/` | grep, sed, awk, cut, sort, uniq, tr, wc |
| 4-5 | File Operations | `bash/06-file-operations/` | find, xargs, tar, rsync, file descriptors, process substitution |
| 5 | Networking | `bash/07-networking/` | curl, ssh, scp, nc, dig, networking scripts |
| 5-6 | Projects | `bash/08-projects/` | Backup script, log analyzer, system monitor, CLI tool |

## Track 3: LaTeX (4-6 weeks)

| Week | Module | Folder | Topics |
|------|--------|--------|--------|
| 1 | Basics | `latex/01-basics/` | Document classes, packages, structure, text formatting |
| 1-2 | Math | `latex/02-math/` | Equations, symbols, matrices, aligned environments |
| 2-3 | Documents | `latex/03-documents/` | Sections, TOC, headers/footers, page layout |
| 3-4 | Tables & Figures | `latex/04-tables-figures/` | Tables, figures, floats, subfigures, wrapping |
| 4 | Bibliography | `latex/05-bibliography/` | BibTeX, natbib, biblatex, citation styles |
| 5 | Presentations | `latex/06-presentations/` | Beamer class, slides, themes, animations |
| 5-6 | Projects | `latex/07-projects/` | Resume, research paper, thesis template, poster |

---

## How to Use This Workspace

1. **Read** the `lesson.md` file in each module folder
2. **Do** the exercises in the exercise files
3. **Check** your work against the solution files (try first!)
4. **Build** the projects to consolidate your skills

## Prerequisites

- A text editor (VS Code recommended)
- Terminal / shell access
- LaTeX distribution (`brew install --cask mactex` or `brew install basictex`)
- Markdown preview (VS Code built-in or any Markdown viewer)
