# Project 3: Create a Markdown Cheat Sheet

## Objective

Create a comprehensive Markdown cheat sheet that you can reference later.
This is both a practice exercise and a useful tool.

## Requirements

Create `markdown-cheat-sheet.md` in this folder covering:

1. All basic syntax (headings, emphasis, lists, links, images, code, blockquotes)
2. Extended syntax (tables, task lists, footnotes)
3. GFM-specific features (alerts, mermaid, math)
4. Tips and common pitfalls

## Format

For each feature, include:
- The rendered description
- The raw syntax in a code block
- A live example (rendered)

## Evaluation Criteria

- [ ] Comprehensive coverage
- [ ] Both syntax and rendered examples
- [ ] Well-organized with table of contents
- [ ] Useful as a quick reference
